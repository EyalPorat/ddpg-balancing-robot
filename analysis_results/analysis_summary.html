Enhanced State Balancing Robot Model Analysis

Analysis of the trained DDPG controller using enhanced state representation. Enhanced state includes:

  • Basic state: theta, theta_dot
  • Action history: Last {0} motor commands
  • Theta history: Last {1} theta values
  • Theta_dot history: Last {2} theta_dot values

All actions are normalized to [-1, 1] range, with motor delay compensation built into the control policy.
